# H1 Посмотри в окно

Сайт позволяет заклянуть в окно разных городов  и насладиться утренним видом солнечной улицы в Краснодаре или полюбоваться закатом в Тбилиси

В этом проекте:
1. Реализована стилизация объектов через css без изменения разметки html
2. Работа с видео
3. Стилизация поля ввода, чекбоксов, кнопок
4. Стилизация состояний объектов в различных состояниях

Язык сайта: русский
